  Skills

  Code

HTML



HTML (Hypertext Markup Language) is a programming language used to create and structure content for the web. It is the foundation of most websites and is used to define the structure and layout of web pages. HTML uses tags to mark up content, such as headings, paragraphs, images, and links.

CSS



CSS (Cascading Style Sheets) is a programming language used to style and format the visual elements of a web page created with HTML. CSS allows designers to separate the presentation of a web page from its content, making it easier to create consistent and visually appealing designs across a website.

Java



Java is a high-level, object-oriented programming language widely used for developing a variety of applications, including web and mobile applications, enterprise software, games, and more. It was created in the mid-1990s by Sun Microsystems and has since become one of the most popular programming languages in the world.

  Office

Word



Microsoft Word is a word processing software that allows users to create, edit, and format various types of documents, such as letters, reports, and resumes. It offers a range of tools and features that make it easy to create professional-looking documents, including templates, fonts, styles, and formatting options.

PowerPoint



Microsoft PowerPoint is a presentation software that enables users to create and deliver dynamic, engaging presentations. It provides a wide range of tools and features that allow users to add text, images, videos, and other multimedia elements to their presentations.

Excel





Microsoft Excel is a spreadsheet software that allows users to organize, analyze, and manipulate data. It offers a range of tools and features that enable users to perform complex calculations, create charts and graphs, and build databases.

  Design

Figma




Figma is a web-based design and prototyping tool that enables users to create and collaborate on interface designs for websites, mobile applications, and other digital products. It offers a range of features that make it easy for designers to create high-fidelity prototypes, including vector networks, responsive design tools, and real-time collaboration.
